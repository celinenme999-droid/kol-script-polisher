# 📝 脚本修改工作流

一个专为广告公司执行人员设计的AI驱动脚本修改工具，支持批量修改视频脚本，根据达人风格和产品Brief自动优化内容。

## ✨ 功能特点

- 📤 **Excel批量处理** - 支持上传Excel文件，批量修改脚本
- 👤 **达人风格分析** - 自动抓取并分析达人账号主页内容
- 🎯 **智能脚本修改** - 根据产品Brief和达人风格自动优化脚本
- 🧠 **记忆功能** - 记录历史修改，避免重复内容
- 🌐 **网页部署** - 一键生成网页链接，方便团队使用
- 📊 **实时预览** - 修改前后对比，实时查看效果

## 🚀 快速开始

### 1. 环境要求

- Python 3.8 或更高版本
- pip 包管理器

### 2. 安装依赖

```bash
# 克隆或下载项目
cd script-modifier-workflow

# 安装依赖包
pip install -r requirements.txt

# 安装Playwright浏览器
playwright install chromium
```

### 3. 运行应用

```bash
# 本地运行
streamlit run app.py
```

应用将在浏览器中自动打开：`http://localhost:8501`

## 📖 使用说明

### 步骤1：上传脚本Excel

1. 点击"选择Excel文件"按钮
2. 选择包含原始脚本的Excel文件
3. 系统会自动读取并预览数据
4. 选择包含脚本内容的列

### 步骤2：达人账号分析

1. 输入达人账号主页链接（支持抖音、小红书、微博等）
2. 点击"分析达人风格"按钮
3. 系统会自动抓取并分析达人内容风格

### 步骤3：填写产品Brief

详细描述产品植入的具体要求，例如：
- 产品出现的时间点
- 口播词的要求
- 需要展示的场景
- 产品的核心卖点

### 步骤4：调整修改设置

- **创意程度**：1-10，1为保守修改，10为大胆创新
- **语气调整**：保持原样/更活泼/更正式/更幽默/更感性
- **长度控制**：保持原长度/适当缩短/适当延长

### 步骤5：开始修改

点击"开始修改脚本"按钮，系统会自动处理所有脚本。

### 步骤6：下载结果

修改完成后，点击"下载修改后的Excel"按钮获取结果文件。

## 💡 使用技巧

1. **首次使用**：建议先测试1-2条脚本，确认效果后再批量处理
2. **Brief编写**：越详细的产品要求，修改效果越好
3. **达人分析**：确保达人账号链接正确，以便准确分析风格
4. **历史记录**：侧边栏会显示最近5次修改记录，方便回顾
5. **批量处理**：建议每次处理不超过50条脚本，以保证质量

## 🐛 常见问题

**Q: 为什么达人分析失败？**
A: 请检查达人账号链接是否正确，某些平台可能有反爬虫机制。

**Q: 修改后的脚本不符合预期？**
A: 尝试调整创意程度和语气设置，或者完善产品Brief。

**Q: 如何批量处理大量脚本？**
A: 建议分批处理，每批不超过50条，以保证修改质量。

**Q: API调用失败怎么办？**
A: 检查API Key是否正确，网络连接是否正常，API额度是否充足。
