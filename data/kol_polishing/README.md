## 📝 软性要求要点

### 开头切入
- 结合打印场景，突出对"稳"的需求
- 从家庭打印痛点切入
- 必须关联创新6重维稳技术

### 6重维稳技术顺序
1. 打印品质稳
2. 输出稳
3. 印量稳
4. 连接稳
5. 适配稳
6. 维护稳

### 禁用词
- 绝对、最好、第一、顶级、必买
