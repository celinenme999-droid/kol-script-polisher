# 🎯 快速入门指南

5分钟快速上手脚本修改工作流！

---

## 📦 第一步：安装（2分钟）

### Mac/Linux用户

```bash
# 进入项目目录
cd script-modifier-workflow

# 运行启动脚本（自动安装所有依赖）
./start.sh
```

### Windows用户

```bash
# 进入项目目录
cd script-modifier-workflow

# 双击运行 start.bat
```

### 手动安装（如果自动脚本失败）

```bash
# 创建虚拟环境
python3 -m venv venv

# 激活虚拟环境
# Mac/Linux:
source venv/bin/activate
# Windows:
venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt

# 安装浏览器
playwright install chromium

# 启动应用
streamlit run app.py
```

---

## 🔑 第二步：配置API Key（1分钟）

1. 浏览器会自动打开应用：`http://localhost:8501`
2. 在左侧边栏找到 "API配置"
3. 选择AI模型提供商（推荐OpenAI）
4. 输入API Key

**获取API Key：**
- OpenAI：https://platform.openai.com/api-keys
- 通义千问：https://dashscope.aliyun.com/apiKey
- Claude：https://console.anthropic.com/

---

## 📤 第三步：上传脚本（30秒）

1. 点击 "选择Excel文件" 按钮
2. 选择包含脚本的Excel文件
3. 系统会自动读取并显示预览
4. 选择包含脚本内容的列

**没有Excel文件？**
使用项目中的示例模板：`templates/示例脚本模板.xlsx`

---

## 👤 第四步：分析达人（30秒）

1. 输入达人账号主页链接
   - 抖音：https://www.douyin.com/user/...
   - 小红书：https://www.xiaohongshu.com/user/...
   - 微博：https://weibo.com/...
2. 点击 "分析达人风格" 按钮
3. 等待分析完成（约10-30秒）

---

## 📋 第五步：填写Brief（1分钟）

在 "产品植入要求/Brief" 文本框中详细描述要求，例如：

```
1. 产品需要在视频前30秒出现
2. 口播词要自然，不要生硬
3. 需要展示产品使用场景
4. 强调产品的核心卖点：温和、不刺激
5. 避免使用夸张的词汇
```

---

## ⚙️ 第六步：调整设置（可选）

根据需要调整以下设置：

- **创意程度**：1-10（1=保守，10=创新）
- **语气调整**：保持原样/更活泼/更正式/更幽默/更感性
- **长度控制**：保持原长度/适当缩短/适当延长

---

## 🚀 第七步：开始修改（1分钟）

点击 "开始修改脚本" 按钮，系统会自动处理所有脚本。

**处理时间：**
- 1-10条脚本：约1-2分钟
- 10-50条脚本：约5-10分钟

---

## 📥 第八步：下载结果（30秒）

修改完成后：
1. 查看修改结果预览
2. 点击 "下载修改后的Excel" 按钮
3. 保存文件到本地

---

## ✅ 完成！

恭喜您已经成功使用脚本修改工作流！

## 💡 使用技巧

### 1. 首次使用建议
- 先测试1-2条脚本
- 确认效果满意后再批量处理

### 2. Brief编写技巧
- 越详细越好
- 明确产品出现的时间点
- 说明需要强调的卖点
- 列出禁止使用的词汇

### 3. 达人分析技巧
- 确保链接正确
- 选择活跃的达人账号
- 分析结果会显示在页面上

### 4. 批量处理建议
- 每批不超过50条
- 分批处理以保证质量
- 定期保存结果

## ❓ 遇到问题？

### 问题1：安装失败
**解决方案：**
- 确保Python版本 >= 3.8
- 检查网络连接
- 尝试手动安装

### 问题2：API调用失败
**解决方案：**
- 检查API Key是否正确
- 确认网络连接正常
- 检查API额度是否充足

### 问题3：达人分析失败
**解决方案：**
- 检查达人链接是否正确
- 某些平台可能有反爬虫机制
- 尝试手动输入达人风格

### 问题4：修改效果不理想
**解决方案：**
- 完善产品Brief
- 调整创意程度和语气设置
- 尝试不同的AI模型
